# المبرمج - دعم فني للسايبر كافيه (Elmobarmeg Tech Support)

موقع تعريفي عصري لخدمات الدعم الفني وصيانة مقاهي الإنترنت والألعاب (Cyber Cafes). يتميز التصميم بطابع "الجيمنج" مع ألوان نيون وجرافيكس احترافية.

## المميزات (Features)
- **تصميم عصري:** واجهة مستخدم داكنة مع تأثيرات نيون (Neon Blue & Purple).
- **متجاوب (Responsive):** يعمل بكفاءة على جميع الشاشات (موبايل، تابلت، كمبيوتر).
- **تفاعلي:** حركات انسيابية باستخدام `motion`.
- **دعم كامل للغة العربية:** استخدام خط Cairo وتنسيق RTL.
- **تواصل سريع:** أزرار مباشرة للتواصل عبر واتساب.

## التقنيات المستخدمة (Tech Stack)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4.0**
- **Lucide React** (للأيقونات)
- **Motion** (للحركات)
- **Vite** (كأداة بناء وتطوير)

## التشغيل محلياً (Local Setup)

1. قم بتحميل المشروع أو عمل `git clone`.
2. افتح المجلد في الـ Terminal.
3. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
4. ابدأ التشغيل في وضع التطوير:
   ```bash
   npm run dev
   ```

## الرفع على GitHub (GitHub Deployment)

1. قم بإنشاء مستودع (Repository) جديد على GitHub.
2. ارفع الملفات باستخدام الأوامر التالية:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin [رابط_المستودع_بتاعك]
   git push -u origin main
   ```

## النشر (Deployment)
المشروع جاهز للنشر على منصات مثل **Vercel** أو **Netlify** أو **GitHub Pages**. بمجرد ربط المستودع بهذه المنصات، سيتم بناء الموقع ونشره تلقائياً.
